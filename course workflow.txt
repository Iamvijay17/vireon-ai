# AI Course Creation Workflow Prompt

You are an expert Product Designer, UX Designer, and Full Stack Software Architect.

Design a complete AI-powered Course Creation workflow for an application similar to Vireon AI.

## Goal

The system allows instructors to create an online course consisting of multiple videos. Each video is generated independently using AI.

The workflow should be simple, scalable, and asynchronous.

---

# Course Structure

A Course is the top-level object.

Example:

Course

* React Basics

  * Introduction to React
  * Installing React
  * JSX Fundamentals
  * Components
  * Props
  * State
  * Events
  * Hooks
  * Mini Project
  * Course Summary

Each course contains unlimited videos.

Each video has its own generation pipeline.

---

# Technical Requirements

* Existing LM Studio integration must be reused for script generation.
* Existing TTS pipeline must be reused for audio generation.
* All generation steps should be asynchronous using background jobs.
* Every step should be resumable if interrupted.
* Users can regenerate only the failed step without restarting the entire workflow.
* Store all generated assets (scripts, audio, scenes, images, and videos) for future editing.
* Every generation action should update the UI in real time with progress indicators and status changes.

Design this workflow as a production-ready system that can scale to thousands of courses and videos while providing a smooth user experience.
